# AI WORLD Landing Page

This is a premium, interactive landing page for **AI WORLD**, a next-generation AI SaaS platform designed to generate high-fidelity image, hyper-realistic, or stylized artwork in seconds from simple text prompts.

## Key Features
- **Modern Design & Animations**: Sleek dark-mode, cosmic-cyberpunk color system (deep indigo, neon cyan, electric purple), glassmorphism cards, and interactive hover states.
- **Interactive Prompt Playground**: Experience image generation directly on the landing page! Choose preset prompt ideas or edit custom ones, adjust aspect ratios (1:1, 16:9, 9:16), pick styles (Cinematic, Fantasy, Anime, 3D Render), and watch the rendering engine progress from noise to a finished image.
- **Dynamic Billing Cycle**: Toggle between monthly and yearly plans to view active pricing changes and discounts.
- **Responsive Navigation**: Adaptable header and drop-down menu styles optimized for mobile, tablet, and widescreen layouts.
- **Community Art Showcase**: Dynamic filterable gallery of community masterpieces with quick prompt copying functionality.

## Project Structure
- `index.html` - Core SEO-optimized structural markup with custom inline SVGs.
- `style.css` - Custom styling rules, keyframes, transitions, and layout adaptions.
- `script.js` - Dynamic interactions, simulations, and pricing controls.
- `assets/` - High-quality sample image assets generated using AI models.

## How to Run Locally

You can run the project locally using Python's built-in HTTP server:
```bash
python -m http.server 8000
```
Then open your browser and navigate to `http://localhost:8000`.
